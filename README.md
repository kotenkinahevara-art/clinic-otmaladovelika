# Ветеринарная клиника «Все создания от мала до велика»

Продакшен-сайт реальной ветеринарной клиники с онлайн-записью и отправкой заявок в Telegram.

## Роль

`Роль: Frontend Developer (solo)`  
Реализовала весь сайт: страницы, адаптивную верстку, форму записи с валидацией, базовое SEO и подключение отправки заявок через backend-эндпоинт.

## О проекте

Проект решает ключевую бизнес-задачу клиники: быстро переводить посетителей сайта в обращения и записи на приём.

Что сделано:
- структурирован многостраничный сайт с понятной навигацией по услугам;
- реализована форма записи с валидацией полей;
- настроена отправка заявок в Telegram;
- добавлена SEO-база: `title`, `description`, `canonical`, `robots.txt`, `sitemap.xml`, Open Graph.

## Технологии и зачем они использованы

- `React` — компонентная архитектура для быстрого развития интерфейса.
- `React Router` — маршрутизация и отдельные SEO-страницы (`/uslugi`, `/zapis`, `/kontakty` и др.).
- `Vite` — быстрая разработка и сборка production-версии.
- `SCSS` — масштабируемые стили и удобная поддержка UI.
- `Telegram Bot API` — мгновенные уведомления о новых заявках без сложной CRM.
- `Playwright` — e2e-проверки ключевых пользовательских сценариев.

## Запуск проекта

1. Установить зависимости:
```bash
npm install
```
2. Запустить frontend локально:
```bash
npm run dev
```
3. Production-сборка:
```bash
npm run build
npm run preview
```
